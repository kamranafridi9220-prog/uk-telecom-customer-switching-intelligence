## Project Overview

This project analyses customer switching behaviour within the UK telecommunications market using publicly available regulatory data.

The objective is to understand the drivers behind telecom provider switching and to develop structured decision-support insights for customer retention strategy.

The analysis draws upon market statistics published by Ofcom, the UK communications regulator, covering switching behaviour, service satisfaction, and competitive pressures across the telecom sector.

Rather than focusing purely on descriptive statistics, the project demonstrates how market-level data can inform strategic retention frameworks for telecom providers operating in competitive environments.
